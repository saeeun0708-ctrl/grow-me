나를 키워줘

지인들이 골라준 단어로 내 캐릭터를 키우고,
"남이 보는 나"를 확인·자랑하는 바이럴 모바일 웹

PRD v1.0 · 2026-06-05

1 제품 개요

"친구들 눈에 비친 진짜 내 모습을, 귀여운 캐릭터 하나로 보여줘."
  • 사용자(주인)는 자신만의 공유 링크를 받는다.
  • 지인이 들어와 이름을 적고, 어울리는 제시어 7개를 선택해 먹이주기를 완료한다.
  • 먹이를 받을 때마다 캐릭터가 성장하고, 가장 많이 받은 제시어의 캐릭터로 변신한다.
  • 3단계까지 자라면 독립시키기로 결과를 확정하고, 인스타 자랑용 이미지로 저장한다.

북극성 지표

🚀 K-팩터 ≥ 1 — 결과 이미지 공유 → 본 사람이 "나도 해볼래" → 신규 가입 루프.

2 타겟 & 핵심 동기

주 타겟10대 후반~20대 (SNS 자기표현·테스트 콘텐츠 친숙)
핵심 동기"남들이 나를 어떻게 볼까?" 호기심 + 자랑·공유 욕구
사용 맥락인스타 스토리, 카톡 단톡방, 친구들 사이 유행

3 핵심 사용자 흐름

👤 주인 (Owner)

1링크 접속 → 간편 로그인 → 내 캐릭터 생성 (1단계, 작은 알)
2내 공유 링크 복사 → 카톡/스토리/DM으로 뿌리기
3지인이 먹이 줄 때마다 성장 / 화면 갱신
43단계 도달 → 원할 때 [독립시키기]
5결과 화면(1~5위 + 받은 수) → 이미지 저장 → SNS 자랑
6(선택) 상세 언락 결제 → 누가 무슨 제시어 골랐는지 전부 확인

🙋 지인 (Sender) — 로그인 불필요, 마찰 최소화

1주인 링크 접속 → "○○님에게 먹이주기" 화면
2내 이름 입력 (예: "단짝 민지")
3제시어 칩 중 7개 선택 (7개 채워야 완료 활성화)
4[먹이주기] → 완료 애니메이션 → "나도 만들기" CTA

8 데이터 모델 (Supabase)

users
  id, auth_provider, provider_user_id, nickname,
  share_slug,        -- 공유 링크용 고유 슬러그
  character_code,    -- 확정 캐릭터 (null=미확정)
  stage,             -- 1 | 2 | 3
  is_independent,    -- 독립 여부
  is_unlocked,       -- 상세 언락 결제 여부
  created_at

feedbacks            -- 먹이 1회 = 1 row
  id, owner_user_id(fk),
  sender_name,       -- 지인 입력 이름
  selected_words,    -- text[] 제시어 7개
  created_at

캐릭터 결정 = selected_words를 카테고리 매핑해 합산 → 최댓값 카테고리. 단계 = feedback 수(0→1, 1~2→2, 3+→3).
